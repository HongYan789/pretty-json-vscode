# Pretty JSON 

## 功能

这是一个json格式化插件，实现将选中内容进行json格式化优雅输出，以及压缩输出json数据

并且同时支持快捷键操作和鼠标右键操作

| 操作         | 快捷键（mac）         | 快捷键（windows） |
| ------------ | --------------------- | ----------------- |
| 优雅输出json | command + j           | cmd + j           |
| 压缩输出json | control + command + j | cmd + ctrl + j    |
